Installation

Follow the steps below to setup your site template:

All Needed Folders And Files

  • assets/fontawesome-free/css/all.min.css Stylesheet files
  • assets/animate.css/animate.min.css Stylesheet files
  • assets/aos/aos.css" Stylesheet files
  • assets/bootstrap/css/bootstrap.min.css Stylesheet files
  • assets/bootstrap-icons/bootstrap-icons.css Stylesheet files
  • assets/glightbox/css/glightbox.min.css Stylesheet files
  • assets/swiper/swiper-bundle.min.css Stylesheet files
  • assets/plyr/plyr.css Stylesheet files
  • assets/purecounter/purecounter_vanilla.js Javascript files
  • assets/aos/aos.js Javascript files
  • assets/bootstrap/js/bootstrap.bundle.min.js Javascript files
  • assets/isotope-layout/isotope.pkgd.min.js Javascript files
  • assets/glightbox/js/glightbox.min.js Javascript files
  • assets/swiper/swiper-bundle.min.js Javascript files
  • assets/plyr/plyr.js Javascript files

You should Keep all or specific files as per your need.

You're now good to go..! Start adding your content/images and generate your beautiful brand new website for your awesome users.

<script>
var elem = document.querySelector('.grid');
var iso = new Isotope( elem, {
    // options
  itemSelector: '.grid-item',
  layoutMode: 'fitRows'
});

// element argument can be a selector string
// for an individual element

    // options
});
</script>
